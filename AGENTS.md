# AGENTS.md — Codex, читай это (правила экосистемы STAW)

Над этим монорепо **по очереди** работают два ИИ-агента — **Codex** и **Claude** — и
владелец проекта. Вы одна команда: один вектор, один общий опыт. Главное:
**ничего не теряется между сменами, и вы не повторяете ошибок друг друга.**

> Полные правила и контекст — в `CLAUDE.md` (корень репозитория). Он и этот файл —
> зеркала. **Прочитай `CLAUDE.md` перед работой.** Ниже — критичный минимум.

## Единый источник правды (вести ОБЯЗАТЕЛЬНО)
- **`docs/WORKLOG.md`** — что сделано / что пробовали / что не вышло / что дальше.
  Читать ПЕРЕД работой, дополнять ПОСЛЕ.
- **`docs/DECISIONS.md`** — принятые решения (не переобсуждать без причины).
- **`docs/PITFALLS.md`** — грабли и тупиковые попытки. **Не повторять то, что тут записано.**
- **`docs/MODULES.md`** — карта модулей экосистемы (что/где/статус). Меняешь модуль — обнови статус.

## Перед работой
1. `git checkout main && git pull`.
2. Прочитать свежий `docs/WORKLOG.md` и весь `docs/PITFALLS.md`.
3. Проверить `git status` и ветку.

## Во время работы
- `main` ЗАЩИЩЁН — прямой push запрещён. Только ветка `feature/*` `fix/*` `docs/*` `chore/*` → **Pull Request** с зелёным CI.
- Контракты API (`auth`/`profile`/`loyalty`) держать стабильными — их используют все приложения.
- **Дизайн/бренд НЕ трогать** — не утверждено. Сейчас важна только корректность и работа экосистемы без багов.
- Репозиторий **ПУБЛИЧНЫЙ** — никогда не коммитить секреты/PII (`.mcp.json`, `*.db` в `.gitignore`).

## После работы
- Дополнить `docs/WORKLOG.md` (дата — Codex — сделано/пробовали-не вышло/решения/состояние/дальше).
- Решение → `docs/DECISIONS.md`; грабли → `docs/PITFALLS.md`. Влить PR (squash).

## Окружение
- Windows. Кириллица в PowerShell бьётся — UTF-8 писать через файловый инструмент.
- Бэкенд (общий, FastAPI+SQLite, dev): `cd backend && PYTHONUNBUFFERED=1 python -m uvicorn main:app --host 0.0.0.0 --port 8000`. Нестабилен — проверяй `/v1/health`.
- Телефон↔бек: `adb reverse tcp:8000 tcp:8000`. GitHub: `Smallfoi/kvartal-ecosystem` (public).

## Состав экосистемы
`kvartal-app/` (Квартал, Flutter) · `sport_store/` (Store, Flutter) · `САЙТ STAW/` (сайт) ·
`backend/` (общий FastAPI). Единый аккаунт (SSO по телефону) + общий баланс баллов. Контракт — `ECOSYSTEM_API.md`.
