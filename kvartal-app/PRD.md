# КВАРТАЛ — PRD (Product Requirements Document)
### Городское беговое соревновательное приложение для Якутска
**Версия:** 1.1 | **Дата:** 2026-05-23 | **Статус:** Confirmed
**Разработчик:** Solo (Python/Go/Java/PHP background)
**Стек:** Flutter (Dart) + Go backend + PostgreSQL + PostGIS
**Платформы:** iOS + Android одновременно

---

## 1. ИССЛЕДОВАНИЕ РЫНКА

### 1.2 Незакрытые ниши на рынке

1. **Русскоязычное** приложение с нативной локализацией — ни один лидер рынка не заточен под РФ/СНГ
2. **Якутск-специфика** — экстремальный холод (-50°C), уникальная городская среда, якутская культура
3. **Совмещение бега + территорий + социала** — все три компонента в одном приложении отсутствуют
4. **Команды по районам** — Strava есть клубы, но нет территориального захвата
5. **Оффлайн-режим** — критично для зон с плохой связью в Якутске

### 1.3 Рыночная возможность

- Якутск: ~330 000 жителей, активная молодёжь, сильная беговая культура
- Рынок спортивных приложений РФ растёт +18% в год (2024)
- Беговые клубы Якутска: Yakutia Runners, Arctic Running, и др. — потенциальные первые партнёры
- Перспектива масштабирования на другие города Сибири/ДВ

## 2. ВИДЕНИЕ ПРОДУКТА

### 2.1 Проблема

Любители бега в Якутске не имеют единой платформы, где можно:
- соревноваться с другими бегунами города в реальном времени
- видеть, как твой клуб «владеет» районами города
- участвовать в уникальных якутских вызовах (бег в мороз, арктические маршруты)

### 2.2 Решение

**КВАРТАЛ** — мобильное приложение, объединяющее:
- GPS-трекинг бега с соревновательными механиками
- Интерактивную карту Якутска с захватом территорий
- Систему клубов/команд и индивидуальных рейтингов
- Геймифицированные вызовы и достижения

### 2.3 Уникальное торговое предложение (USP)

> *«Сделай Якутск своим — бегай, захватывай районы, становись легендой города»*

## 4. ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ

### 4.1 Модуль 1 — GPS-трекер бега (CORE)

**F1.1 Запись тренировки**
- Запуск/пауза/стоп трекинга
- Отображение: расстояние, темп, время, скорость, каденс (шаги/мин)
- Маршрут на карте в реальном времени
- Поддержка фонового режима (экран заблокирован)

**F1.2 История тренировок**
- Список всех пробежек с метриками
- График прогресса (неделя/месяц/год)
- Сравнение с предыдущими пробежками

**F1.3 Цели**
- Дневная/недельная цель по км
- Уведомления о достижении цели

### 4.2 Модуль 2 — Захват территорий (HERO FEATURE)

**F2.1 Карта Якутска**
- Карта города разбита на **гексагоны** (H3-сетка, ~300м в диаметре)
- Каждый гексагон можно «захватить» пробежав через него
- Цвета: нейтральный (серый) / твой (цвет клуба) / чужой

**F2.2 Механика захвата**
- Бегун захватывает гексагон, если пробежал через него ≥50м
- Территория удерживается **72 часа**, затем угасает (нужно подтверждать)
- Чем больше раз ты пробегал — тем «сильнее» захват (1–5 звёзд)
- Клубный захват: если 3+ члена клуба пробежали — территория становится клубной

**F2.3 Карта города**
- Тепловая карта активности города
- Топ захваченных зон недели
- «Спорные» гексагоны (несколько претендентов)

**F2.4 Районы Якутска**
- Специальные «районные боссы» — контрольные точки в каждом районе (КТ)
- Кто владеет КТ — владеет районом
- Районы: Октябрьский, Строительный, Сергелях, Табага, Птицефабрика, и др.

### 4.3 Модуль 3 — Соревнования и вызовы

**F3.1 Еженедельные вызовы**

| Вызов | Описание | Призы |
|---|---|---|
| 🏃 Марафонец недели | Кто пробежит больше всех км за неделю | Золотой бейдж |
| ❄️ Арктический бегун | Пробежать ≥5 км при -30°C и ниже | Уникальный аватар |
| 🗺️ Исследователь | Открыть 10 новых гексагонов за неделю | Очки территории |
| ⚡ Спринтер | Лучший темп на 1 км в городе | Звание «Быстрейший» |
| 🌅 Ранняя пташка | Пробежать до 7:00 утра 5 раз | Рамка профиля |
| 🌙 Ночной бегун | Пробежать после 22:00 | Ночной бейдж |
| 🔥 Серия | Не пропустить 7 дней подряд | Огненный трейл |
| 🏔️ Высотник | Набрать 500м+ перепада высот | Горный титул |

**F3.2 Сезонные чемпионаты**
- Месячный чемпионат по дистанции (инд. + команды)
- Сезонный захват карты (кто владеет максимумом районов)
- Зимний ультра-вызов (декабрь–февраль, Якутск -50°C)
- «Белые ночи» — летний нон-стоп марафон

**F3.3 Дополнительные форматы соревнований**

| Тип | Механика |
|---|---|
| **Гонка призраков** | Соревнуешься с записанным маршрутом лучшего бегуна на этой дороге |
| **Эстафета** | Команда 4 чел., каждый бежит свой отрезок, передаёт виртуальную эстафету |
| **Городской квест** | Серия КП по карте Якутска, нужно добраться до всех за минимум времени |
| **Бег вслепую** | Маршрут открывается по 100м вперёд — куда бежать, узнаёшь на ходу |
| **Битва кланов** | Два клуба соревнуются суммарной дистанцией за 48 часов |
| **Контрольные точки** | Добежать до 5 случайных точек в городе за минимум времени |

### 4.4 Модуль 4 — Профиль и социал

**F4.1 Профиль пользователя**
- Аватар + рамка (разблокируется за достижения)
- Статистика: всего км, гексагонов, побед
- Тепловая карта личных пробежек по городу
- Коллекция бейджей/достижений
- Звание (Новичок → Бегун → Атлет → Легенда → Квартал Якутска)

**F4.2 Клубы**
- Создать/вступить в клуб (макс. 50 чел.)
- Клубный рейтинг и лидерборд
- Клубный цвет на карте территорий
- Клубный чат (базовый)
- Клубные вызовы внутри команды

**F4.3 Лента активности**
- Фид пробежек друзей/клуба
- Лайки и комментарии к тренировкам
- Поделиться пробежкой в соцсети

**F4.4 Рейтинги**
- Глобальный рейтинг Якутска
- Рейтинг по районам
- Рейтинг клубов
- Еженедельный топ-10

### 4.5 Модуль 5 — Якутск-специфика

**F5.1 Погодная интеграция**
- Текущая температура на экране бега
- «Морозный коэффициент»: бонус к очкам при беге в мороз (чем холоднее — тем больше)
- Предупреждения о критическом морозе (-45°C и ниже)

**F5.2 Арктические маршруты**
- База рекомендованных маршрутов по Якутску
- Зимние маршруты с пометкой «проверено сообществом»
- Маршруты по набережной Лены, Центру, паркам

**F5.3 Локальные события**
- Расписание городских забегов
- Интеграция с якутскими беговыми клубами
- Push-уведомления о местных ивентах

## 5. НЕФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ

### 5.1 Производительность
- Запуск трекера: ≤2 сек
- GPS-фиксация: ≤5 сек (в городских условиях)
- Обновление карты территорий: real-time (<500ms задержка)
- Работа в оффлайн: трекер + кэшированная карта (без соцфункций)
- Поддержка слабого интернета (2G/Edge режим)

### 5.2 Точность GPS
- Фильтрация GPS-шума (алгоритм сглаживания маршрута)
- Защита от читерства: блокировка записи при скорости >40 км/ч
- Калибровка под плотную городскую застройку Якутска

### 5.3 Энергопотребление
- Оптимизация батареи (критично при -50°C, батарея садится быстрее)
- Режим экономии: сниженная частота GPS при долгих пробежках
- Уведомление при заряде <15% во время тренировки

### 5.4 Платформы
- iOS 15+ (iPhone 8 и новее)
- Android 8.0+ (API 26+)
- Фреймворк: **React Native + Expo** (кроссплатформа) или **Flutter**
- Поддержка планшетов: нет (только телефоны)

### 5.5 Масштабируемость
- Архитектура: готова к масштабированию на другие города РФ
- БД: поддержка 50 000+ активных пользователей
- CDN для карт: быстрая загрузка тайлов

## 6. ДИЗАЙН И UX

### 6.1 Дизайн-философия

**«Арктический драйв»** — быстрый, энергичный, холодно-красивый дизайн:
- Тёмная тема по умолчанию (Dark mode first)
- Акцентный цвет: **электрик-синий** (#0A84FF) + **ледяной белый** (#E8F4FD)
- Фоновые паттерны: геометрия якутских орнаментов
- Типографика: жирная, читается на бегу
- Анимации: плавные, 60fps, ощущение скорости

### 6.2 Ключевые экраны

```
Главный экран
├── Карта (80% экрана) — территории в реальном времени
├── Быстрый запуск тренировки (FAB-кнопка)
└── Мини-статистика (шаги сегодня, км недели)

Экран тренировки
├── Большой таймер + дистанция
├── Живой маршрут на карте
├── Захватываемые гексагоны (подсвечиваются при пробеге)
└── Кнопки: пауза / стоп

Карта территорий
├── Полноэкранная карта Якутска
├── Гексагональная сетка с цветами клубов
├── Фильтры: мой клуб / все / по районам
└── Leaderboard боковая панель

Профиль
├── Тепловая карта личных пробежек
├── Достижения и бейджи
└── Клубный статус

Соревнования
├── Активные вызовы
├── Таблица лидеров
└── Предстоящие события
```

### 6.3 Навигация

5 вкладок в нижнем табборе:
1. 🗺️ **Карта** — территории города
2. 🏃 **Бег** — запуск тренировки
3. 🏆 **Рейтинг** — лидерборды
4. 👥 **Клуб** — команда и чат
5. 👤 **Профиль** — личная статистика

## 7. ТЕХНИЧЕСКИЙ СТЕК

### 7.1 Frontend (Mobile)
```
Flutter 3.x (Dart)
├── Maps: flutter_map + OpenStreetMap (бесплатно)
├── Hex grid: h3_flutter (Uber H3 Dart port)
├── GPS: geolocator + background_locator_2 (фоновый режим)
├── Health: health (Apple Health + Google Fit)
├── BLE sensors: flutter_blue_plus (HR-мониторы, v2.0)
├── State: Riverpod 2.x (рекомендован для Flutter)
├── Navigation: go_router
├── Charts: fl_chart
└── Animations: flutter_animate + Lottie
```

### 7.2 Backend
```
Go (Fiber v2 или Gin)
├── БД: PostgreSQL + PostGIS (геоданные)
├── Real-time: WebSocket (gorilla/websocket) для карты
├── Cache: Redis (лидерборды, территории)
├── Auth: JWT + SMS (smsc.ru API) + VK OAuth + Google + Apple
├── Storage: Yandex Object Storage (S3-совместимый)
└── Maps tile: OpenStreetMap (tile.openstreetmap.org или self-hosted)
```

### 7.3 Инфраструктура
```
Хостинг: Яндекс.Облако (для скорости в РФ/ДВ)
├── Сервер: Владивосток или Новосибирск (ближайшие к Якутску)
├── CDN: Яндекс CDN
├── Monitoring: Grafana + Prometheus
└── CI/CD: GitHub Actions + Expo EAS Build
```

### 7.4 Карты
- **OpenStreetMap** + собственный стиль (тёмная тема)
- **H3 (Uber Hexagonal Hierarchy)** — гексагональная сетка территорий
- Разрешение H3 уровня 9 (~174м) для городских гексагонов
