# STAW Official Store Site

Стартовый проект сайта для официального магазина китайского спортивного бренда STAW / 世拓.

## Что внутри

- `index.html` - главная страница магазина
- `styles.css` - адаптивный дизайн и анимации
- `script.js` - reveal-анимации, фильтры каталога и демо-корзина
- `PROJECT.md` - описание проекта и дальнейший план
- `BRAND_CONTEXT.md` - постоянный конспект по бренду
- `HISTORY.md` - журнал изменений
- `Референсы/` - папка для будущих фото, видео и образцов товаров

## Как открыть

Откройте `index.html` в браузере. Проект статический и не требует dev server.

## Направление

Сайт задуман как премиальный product-story для urban running / light outdoor бренда: крупные визуальные сцены, плавные scroll reveal-анимации, ясная философия, каталог и демо-корзина.

В будущем в `Референсы/Видео` можно добавлять фоновые ролики для hero, деталей ткани, движения модели и outdoor-сцен.

## Единая экосистема проектов

Этот проект не является отдельным продуктом. Он является частью общей экосистемы:

- `bugun-app` / «Квартал» — мобильное приложение для бегунов: захват территорий, баллы, клубы, соревнования.
- `САЙТ STAW` — сайт бренда/магазина, который далее будет переименован под общий бренд.
- `sport_store` — мобильное приложение спортивного магазина.

Перед началом работы обязательно прочитать `RECOMMENDATION.md`. Это общее рекомендательное письмо для всех трёх проектов: в нём собраны выводы по Nike, Adidas, Puma, Gymshark, Under Armour и описана стратегия общей экосистемы, а не изолированных приложений.

Общий смысл: приложения и сайт должны развиваться как связанная система — единый бренд, единые пользователи, заказы, лояльность, баллы, уведомления, контент и будущая backend/admin архитектура.
