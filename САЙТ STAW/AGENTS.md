# STAW / бренд-сайт — AGENTS.md

## Важно: единая экосистема

Этот сайт не является отдельным продуктом. Он часть общей экосистемы вместе с:

- `bugun-app` / «Квартал» — мобильное приложение для бегунов: захват территорий, баллы, клубы, соревнования.
- `sport_store` — мобильное приложение спортивного магазина.
- `САЙТ STAW` — сайт бренда/магазина, который далее будет переименован под общий бренд.

Перед началом работы обязательно прочитать `RECOMMENDATION.md`. Это общее рекомендательное письмо для всех трёх проектов: анализ Nike, Adidas, Puma, Gymshark, Under Armour и рекомендации по общей продуктовой стратегии.

Работая над сайтом, учитывать связь с мобильными приложениями: единый бренд, единая аудитория, лояльность, баллы, заказы, уведомления, каталог, admin/backend контур.
