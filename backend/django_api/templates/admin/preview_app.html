Превью приложения

Реальные экраны приложения (Flutter Web, режим PREVIEW=1 — с черновиками). Правьте товары/баннеры → «Обновить».

Открыть в новой вкладке

Если превью пустое — соберите web-превью приложения (dev):

cd sport_store
flutter build web --pwa-strategy=none --dart-define=PREVIEW=1 \
  --dart-define=SPORT_STORE_API_BASE_URL=http://localhost:8000/v1 --no-tree-shake-icons
cd build/web && python -m http.server 5578

Адрес меняется переменной APP_PREVIEW_URL (для прод-хостинга сборки).
